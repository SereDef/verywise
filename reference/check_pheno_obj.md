# Check that object exists in the global environment

Check that object exists in the global environment

## Usage

``` r
check_pheno_obj(obj_name)
```

## Arguments

- obj_name:

  A character string specifying the name of the object

## Value

The object from the global environment (if it exists).

## Details

This function checks if the specified object exists in the global
environment. It returns the object if found, otherwise raises an error
with a message.
