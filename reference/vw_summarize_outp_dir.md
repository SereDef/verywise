# Summarise output directory: measures × hemispheres per subdirectory

Summarise output directory: measures × hemispheres per subdirectory

## Usage

``` r
vw_summarize_outp_dir(outp_dir)
```

## Arguments

- outp_dir:

  Path to the top-level output directory

## Value

Invisibly returns a named list: one element per subdir, each containing
a named list of measures → character vector of hemispheres
