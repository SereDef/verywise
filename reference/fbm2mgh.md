# Save file backed matrix (FBM) to MGH file

Save file backed matrix (FBM) to MGH file

## Usage

``` r
fbm2mgh(fbm, fnames, mode = "1row.1file")
```

## Arguments

- fbm:

  a file backed matrix

- fnames:

  file name to be used to save out the data

- mode:

  "1row.1file" or "allrows.1file"
