# Print pretty messages to console

Adds fillers to messages

## Usage

``` r
pretty_message(content, fill = "=", tot_nchar = 60, center = TRUE)
```

## Arguments

- content:

  : The message to print

- fill:

  : (default "=")

- tot_nchar:

  : (default: 60)

- center:

  : (default: TRUE) center the content.

## Value

A prettyfyed console message.
